# Python Regex Examples

This folder contains Python code examples demonstrating regex patterns from all categories.

## Structure

```
python/
├── README.md
├── 01_basics/
│   ├── simple_patterns.py
│   ├── anchors_literals.py
│   └── basic_exercises.py
├── 02_intermediate/
│   ├── character_classes.py
│   ├── grouping_capturing.py
│   ├── lookarounds.py
│   └── intermediate_exercises.py
├── 03_advanced/
│   ├── backreferences.py
│   ├── greedy_lazy.py
│   └── complex_patterns.py
└── 04_projects/
    ├── email_validator.py
    ├── log_parser.py
    └── html_scraper.py
```

## Usage

Run any Python file to see the examples in action:

```bash
# Basic examples
python python/01_basics/simple_patterns.py

# Intermediate examples
python python/02_intermediate/character_classes.py

# Advanced examples
python python/03_advanced/backreferences.py

# Projects
python python/04_projects/email_validator.py user@example.com
```

## Requirements

- Python 3.6+
- No external dependencies (uses built-in `re` module)

## Learning Path

1. Start with `01_basics/` to understand fundamental patterns
2. Move to `02_intermediate/` for more complex patterns
3. Explore `03_advanced/` for advanced techniques
4. Build projects in `04_projects/` to apply your knowledge

**Note:** These examples correspond to the markdown documentation in the parent directories.
