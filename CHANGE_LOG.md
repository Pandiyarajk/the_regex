# Change Log

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-11-16

### Added
- Initial release of Regex Mastery guide (Version 1.0)
- Initial commit by Pandiyaraj Karuppasamy
- Complete learning path from basics to advanced
- 01-basics module with introduction, syntax, examples, exercises, and quiz
- 02-intermediate module with character classes, grouping, lookarounds, examples, exercises, and quiz
- 03-advanced module with backreferences, greedy/lazy matching, complex examples, exercises, and quiz
- 04-real-world-projects module with 5 practical projects
- Solutions folder with answers for all exercises
- Tools folder with recommended tools and playgrounds documentation
- Comprehensive README.md with learning path
- CHANGE_LOG.md for tracking updates

### Features
- Structured learning path for regex beginners to advanced users
- Hands-on examples with explanations
- Practice exercises at each level
- Quizzes to test understanding
- Real-world mini projects
- Multi-language support (Python and JavaScript)
- Complete solutions for all exercises
